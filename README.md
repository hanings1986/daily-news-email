# 综合信息分析系统

## 🎯 项目简介

这是一个自动化的综合信息分析系统，包含两个主要功能：
- **国际新闻摘要**: 每天早上7:00自动整合主流媒体的国际新闻，生成摘要报告
- **竞彩足球分析**: 每天下午16:45自动整合主流媒体的赛事分析和推荐结果

## ✨ 主要功能

### 国际新闻摘要功能
- **定时摘要**: 每天7:00自动执行新闻摘要任务
- **多源数据**: 整合新华网、人民网、央视新闻、BBC中文、路透中文等主流媒体
- **智能分类**: 自动按国际、财经、科技等类别分类新闻
- **重要性评估**: 自动评估新闻重要性并生成重点新闻列表
- **趋势分析**: 分析当日新闻趋势和关注焦点

### 竞彩足球分析功能
- **定时分析**: 每天16:45自动执行分析任务
- **多源数据**: 整合腾讯体育、网易体育、新浪体育、虎扑体育等主流媒体分析
- **智能整合**: 自动分析媒体共识和推荐趋势
- **多种报告**: 生成Markdown和Excel格式的详细报告

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行系统

#### 方式一：立即执行一次新闻摘要
```bash
python main.py --news-once
```

#### 方式二：立即执行一次竞彩足球分析
```bash
python main.py --run-once
```

#### 方式三：启动定时服务（推荐）
```bash
python main.py --start-service
```

#### 方式四：交互式运行
```bash
python main.py
```

### 3. 查看报告

报告将生成在 `reports/` 目录下：

**新闻摘要报告**:
- `国际新闻摘要_YYYYMMDD.md` - Markdown格式报告
- `国际新闻摘要_YYYYMMDD.xlsx` - Excel格式报告（需要pandas）

**竞彩足球分析报告**:
- `竞彩足球分析报告_YYYYMMDD.md` - Markdown格式报告
- `竞彩足球分析报告_YYYYMMDD.xlsx` - Excel格式报告（需要pandas）

## ⚙️ 配置说明

在 `config.py` 中可以修改以下配置：

### 定时任务配置
- `NEWS_SCHEDULE_TIME`: 新闻摘要执行时间（默认7:00）
- `FOOTBALL_SCHEDULE_TIME`: 竞彩足球分析执行时间（默认16:45）

### 新闻源配置
- `NEWS_DATA_SOURCES`: 新闻数据源配置，包含国际、财经、科技等类别

### 分析配置
- `MAX_NEWS_ITEMS`: 最大新闻条目数量（默认15条）
- `HOT_MATCH_THRESHOLD`: 热门赛事阈值
- `MAX_MATCHES`: 最大显示赛事数量

## 📊 报告内容

### 新闻摘要报告内容
1. **今日新闻概览** - 统计总新闻数、平均重要性等
2. **新闻类别分布** - 按国际、财经、科技等类别统计
3. **新闻来源分布** - 各媒体来源统计
4. **今日重点新闻** - 重要性最高的3条新闻详细分析
5. **详细新闻列表** - 所有新闻的表格形式展示
6. **今日新闻趋势分析** - 分析当日主要关注领域
7. **风险提示** - 信息使用注意事项

### 竞彩足球分析报告内容
1. **今日热门赛事概览** - 表格形式展示热门赛事
2. **详细赛事分析** - 每场比赛的媒体观点和推荐
3. **投资建议总结** - 推荐结果的统计分析
4. **风险提示** - 投资风险提醒

## 🔧 自定义扩展

### 添加新的新闻源
在 `news_collector.py` 中的 `get_mock_news()` 方法添加新的新闻数据，或实现真实的数据采集逻辑。

### 修改新闻分析逻辑
在 `news_collector.py` 中的 `analyze_news_trends()` 方法修改分析算法。

### 调整报告格式
在 `news_generator.py` 中修改新闻摘要报告生成逻辑。

## ⚠️ 注意事项

1. **数据来源**: 当前版本使用模拟数据，实际使用时应替换为真实的数据采集逻辑
2. **反爬虫**: 采集真实网站数据时需遵守robots.txt和相关法律法规
3. **风险提示**: 本系统仅供分析参考，不构成投资建议
4. **定时任务**: 确保程序持续运行才能保证定时任务正常执行
5. **信息准确性**: 新闻信息具有时效性，建议结合其他信息源进行验证
